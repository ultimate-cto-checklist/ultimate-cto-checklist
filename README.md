# Ultimate CTO Checklist

A collection of best practices, guides, and scripts for technical projects from a CTO perspective.

## Purpose

- **Verify** that essential elements exist in any technical project
- **Guide** teams to set things up if they're missing
- **Automate** where possible with scripts that can be run via Clawdbot or other AI harnesses

## Structure

Items will be organized into:
- **Guides** - Documentation and step-by-step instructions
- **Scripts** - Automated verification and setup tools
- **Skills** - Clawdbot/AI-compatible skills for auditing projects

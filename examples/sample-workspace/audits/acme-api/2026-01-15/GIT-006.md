## Evidence

```bash
$ git branch -r | grep -v HEAD
  origin/develop
  origin/main
  origin/staging
```

Has a `develop` branch in addition to `main` and `staging`.

## Notes

Recommend removing the `develop` branch and merging directly to `main` via PRs. The develop branch adds unnecessary complexity to the branching model.

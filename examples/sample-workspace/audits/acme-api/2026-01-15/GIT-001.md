```bash
$ cd /tmp/test-clone
$ cp .env.example .env
$ docker-compose up -d
$ curl http://localhost:3000/health
{"status":"ok"}
```

Repository clones and runs with only Docker required.

Clean setup. Docker Compose handles all dependencies (Postgres, Redis).

## Evidence

```bash
$ unset DATABASE_URL
$ npm start
Error: connect ECONNREFUSED 127.0.0.1:5432
```

Missing DATABASE_URL causes a raw connection error instead of a helpful message.

## Notes

Recommend adding startup validation that checks for required env vars and prints human-readable messages before attempting connections.

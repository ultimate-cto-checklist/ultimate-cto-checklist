# Available Commands

## Setup
- `/audit-tutorial` - First-time walkthrough
- `/audit-init` - Initialize this workspace
- `/audit-add-project` - Register a project for auditing

## Execution
- `/audit-start <project>` - Begin a new audit
- `/audit-continue` - Resume where you left off
- `/audit-item <id>` - Jump to a specific item
- `/audit-skip <id>` - Skip an item with reason
- `/audit-section <name>` - Focus on one section

## Reporting
- `/audit-status` - View current progress
- `/audit-summary` - Generate full report
- `/audit-history` - View past audits
- `/audit-diff <date1> <date2>` - Compare two audits
- `/audit-waiver <id>` - Create an exception waiver
